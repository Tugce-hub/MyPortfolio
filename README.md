# MyPortfolio

Bu proje, ASP.NET Core MVC ile geliştirilmiş bir portfolyo uygulamasıdır.  

## Özellikler
- Hakkımda sayfası
- Projelerim
- Yetenekler
- Deneyimlerim
- İletişim formu
- Admin Paneli

## Kullanılan Teknolojiler
- ASP.NET Core
- Entity Framework Core
- MSSQL
- Bootstrap 5
- JavaScript
